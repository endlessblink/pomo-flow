Manual Testing Instructions

To test the compact context menu functionality:

  1. Open the application at http://localhost:5174
  2. Navigate to the Board view
  3. Change the density selector to "Ultra-thin" (the first button with minimize icon)
  4. Right-click on any task card to open the context menu
  5. Verify that these menu items show only icons (no text labels):
    • Focus Mode (eye icon)
    • Start Now (play icon)
    • Start Timer (play icon)
    • Duplicate (copy icon)
  6. Switch back to "Comfortable" density and verify text labels reappear

Expected Behavior:

  • Normal Mode: Context menu shows icon + text + shortcut
  • Compact Mode: Context menu shows only icon + shortcut
  • Hover: Both modes should show tooltips for accessibility
