üìã Test Steps

Step 1: Open the Application

Open the app in the iframe below. It should load at http://localhost:5173 (or another available port).

Step 2: Check Console for Initialization

Open browser console (F12) and look for:

  • UndoRedo: Global keyboard handler initialized
  • No import errors or "Undo/Redo system not available" warnings

Step 3: Create a Task

  1. Go to Board view
  2. Create a new task
  3. Check console for UndoRedo: Executed - Create task

Step 4: Test Undo (Ctrl+Z)

  1. Press Ctrl + Z
  2. Task should disappear
  3. Check console for UndoRedo: Undid - Create task
  4. Check console for UndoRedo: Undo shortcut triggered

Step 5: Test Redo (Ctrl+Shift+Z)

  1. Press Ctrl + Shift + Z
  2. Task should reappear
  3. Check console for UndoRedo: Redid - Create task
  4. Check console for UndoRedo: Redo shortcut triggered

Step 6: Test Alternative Redo (Ctrl+Y)

  1. Press Ctrl + Y
  2. Should also redo the action

// Test direct store access
import('/src/stores/undoRedo.js').then(({ useUndoRedoStore }) => {
  const store = useUndoRedoStore();
  console.log('UndoRedo Store:', {
    canUndo: store.canUndo,
    canRedo: store.canRedo,
    undoCount: store.undoCount,
    redoCount: store.redoCount,
    lastAction: store.lastActionDescription
  });
});

// Test keyboard handler
import('/src/utils/globalKeyboardHandler.js').then(({ getGlobalKeyboardHandler }) => {
  const handler = getGlobalKeyboardHandler();
  console.log('Keyboard Handler:', handler ? 'Active' : 'Not found');
  if (handler) {
    console.log('Settings:', handler.getSettings());
  }
});

// Test manual undo/redo
import('/src/stores/undoRedo.js').then(({ useUndoRedoStore }) => {
  const store = useUndoRedoStore();
  console.log('Before undo - Can undo:', store.canUndo);
  store.undo().then(() => {
    console.log('After undo - Can undo:', store.canUndo, 'Can redo:', store.canRedo);
  });
});
